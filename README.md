# Urban Fire Analysis

This repository contains Python code for analyzing urban fire incident data in Chicago.

The goal is to explore fire incident patterns, risk factors, and relationships between fire occurrence, building characteristics, census variables, and other spatial or socioeconomic features.

## Repository Structure

```text
urban-fire/
├── data/          # Local datasets, not uploaded to GitHub
├── notebooks/     # Exploratory analysis notebooks
├── src/           # Python scripts and reusable functions
├── results/       # Generated analysis outputs
├── figures/       # Generated plots
└── README.md
```

## Data

The full CSV datasets are not included in this public repository.

Place the required CSV files inside the data/ folder before running the analysis.
